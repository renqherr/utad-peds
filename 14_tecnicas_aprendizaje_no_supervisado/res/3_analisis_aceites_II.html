Análisis de aceites de oliva (II)

Analizar los clústers construidos sea con k-medias o con k-medioides:

  • ¿Qué variables son las que más distinguen los clústers? ¿Alguna no lo hace?
  • ¿Puedes hacer un pequeño estudio de la calidad del clúster (gráficos de bandera o similar)?
  • Estudiar uno de los clústers y mostrar la distribución de las variables en él. Comparar con la distribución de la muestra entera.
  • ¿Se correlacionan estos clústers con las variables geográficas?

Cargamos los datos:

set.seed(1234)

olive <- read.table(file = "../dat/olive.txt", header = T, sep = "\t")
data <- scale(olive[, -c(1, 2, 11)])

Generemos 6 clústeres para k-means y analizamos el peso de cada variable del dataset para cada clúster usando la función plot_clus_coord del paquete pablo14/clusplus:

fit.km <- kmeans(data, centers = 6, nstart = 25)
aggregate(olive[, 3:10], by=list(cluster=fit.km$cluster), mean)

Observamos que las variables de mayor peso para cada clúster (por encima del 70%) son:

  • Para el C1: palmitic, stearic, linoleic, arachidic y eicosenoic
  • Para el C2: linoleic y arachidic
  • Para el C3: oleic, linoleic y arachidic
  • Para el C4: stearic, oleic, linoleic, arachidic y eicosenoic
  • Para el C5: palmitic, palmitoleic, linoleic, linolenic arachidic y eicosenoic
  • Para el C6: stearic y oleic
